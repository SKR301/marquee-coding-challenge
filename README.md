  ## Prerequisite
  - React.JS or Expo React
  - Node.JS
  - Postgresql
  
  
  ## HowTo
  How to start the program:-
  1. Clone the repo.
  2. `yarn install --ignore-engines`
  3. configure database credentials in ***db.js*** file
  4. `node api.js`
  5. `expo start -web`
